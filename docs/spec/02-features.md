## D. 품질 / 비기능 요구사항

### D1. 성능

- 1MB HWPX 로드 < 1.0초 (파싱 + 초기 렌더)
- 10,000 문단 문서에서 타이핑 시 60fps 유지 (가상 스크롤 적용)
- 번들 크기 < 1.5MB gzipped (폰트 제외)

### D2. 호환성

- 웹 브라우저: Chrome/Edge ≥ 최신 2버전, Firefox ≥ 최신 2버전, Safari ≥ 최신 2버전
- 데스크톱: macOS ≥ 12 (Universal Binary: Apple Silicon + Intel), Windows 10/11 (x64, arm64 best-effort)
- 한컴오피스 2020 이상에서 열림
- 모바일: v1 미지원, v2에서 iOS/Android 정식 지원 (Tauri 2 mobile)

### D3. 접근성

- 키보드 내비게이션 전 기능
- 스크린 리더(NVDA, VoiceOver) 기본 지원
- WCAG 2.1 AA 기준 색 대비

### D4. 보안

- 내장 Scripts 실행 안 함
- 이미지 업로드 시 MIME 검증, SVG 는 sanitize
- XML 파싱 시 외부 엔티티(XXE) 차단
- CSP 기본 권장 설정 문서화

### D5. 국제화 / 지역화

- UI 한국어 기본, 영어 지원
- 한글 IME / 입력기 최우선 테스트 대상
